# DSR Frizerski Salon

Web application for managing a hair salon, including services, products, reservations and invoices.

## Technologies
- PHP
- MySQL
- Docker
- phpMyAdmin
- HTML/CSS/JavaScript

## Features
- User roles
- Reservations
- Services and products
- Invoices
- Google OIDC login
- QR codes
- Email notifications

## How to run locally
docker compose up

Web app:
http://localhost:8081

phpMyAdmin:
http://localhost:8080
